# HWP IT!
HWP의 수식 문법으로 작성된 수식을 포함한 txt파일을 hwp파일로 변환합니다.

## 사용 방법
### 요구사항
- Windows
- 아래아 한글 설치

### TXT 파일 작성
일반 텍스트와 수식을 포함한 `.txt` 파일을 준비합니다.  
수식은 `$수식 내용$`의 형태로 작성합니다.  
수식은 HWPEQN 문법으로 작성합니다.  
  
예)  
`원의 방정식은 $x^{2} + y^{2} = 25$이다.`

### 파일 변환
HWP IT 앱에서 앞서 작성한 TXT파일을 선택하여 `.hwp`파일로 변환합니다.
